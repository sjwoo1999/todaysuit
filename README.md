## 소개

오늘만정장은 면접, 졸업사진, 결혼식 등 중요한 순간을 위한 정장 렌탈 서비스입니다.

> "사회로 나가는 첫걸음이 빚이나 짐이 되어서는 안 됩니다."

### 왜 오늘만정장인가요?

| 구분 | 신품 구매 | 오늘만정장 |
|------|----------|-----------|
| 가격 | 30만원+ | 3.3만원~ |
| 세탁 | 직접 | 포함 |
| 접근성 | - | 캠퍼스 픽업 |

---

## 주요 기능

### 1. 캠퍼스 픽업
학교 정문 픽업 스테이션에서 공강 시간에 바로 수령. 등교길에 픽업, 하교길에 반납.

### 2. AI 사이즈 추천
사진 한 장으로 체형 분석. 딱 맞는 사이즈를 추천받고, 안 맞으면 1회 무료 교환.

### 3. 세탁 포함
입은 그대로 반납. 전문 세탁은 저희가 책임집니다.
